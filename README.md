# Student Performance Data Analysis

## Overview

This project was completed as **Task 1** during my **1-Month Data Analysis Internship at Maincrafts Technology**.

The objective of this project was to perform **Exploratory Data Analysis (EDA)** on a Student Performance dataset and extract meaningful insights using Python.

## Dataset

The dataset contains information about student demographics, study habits, and academic performance.

Key attributes include:

* Gender
* Age
* Study Time
* Absences
* First Period Grade (G1)
* Second Period Grade (G2)
* Final Grade (G3)

## Objectives

* Load and explore the dataset
* Understand dataset structure and data types
* Check for missing values and duplicates
* Analyze student performance
* Compare academic performance by gender
* Visualize grade distributions
* Study the relationship between study time and grades

## Technologies Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Analysis Performed

### Data Cleaning

* Checked dataset structure
* Verified data types
* Checked for missing values
* Identified and removed duplicate records

### Statistical Analysis

* Calculated average final grade
* Identified students scoring above 15
* Compared average performance between male and female students

### Visualizations

* Histograms of grades (G1, G2, G3)
* Scatter plots: Study Time vs Grades
* Bar chart: Male vs Female Average Score

## Key Insights

* Student grades were analyzed across multiple assessment periods.
* Gender-wise performance differences were evaluated.
* Study time showed a relationship with academic performance.
* Visualizations helped identify patterns and trends within the dataset.

## Outcome

This project enhanced my understanding of:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Data Visualization
* Insight Generation using Python
